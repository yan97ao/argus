#### 🟡 中重要度变更 (1)

### docs: expand the dynamo-docs skill and regroup the style-guide docs section (#10772)
**SHA**: `290609f` | 🔗 [查看提交](https://github.com/ai-dynamo/dynamo/commit/290609f8c2e967ec8c4f29d48e71c950857b7ae2)

**🎯 变更类型**：文档/流程增强  
**⚡ 重要程度**：🟡 中  
**📋 变更摘要**  
1. **dynamo‑docs skill** 章节大幅扩展，加入了完整的“Style Guide Is the Standard”“Content Rules”“Operations”“Redirects & version model”等细则，明确了 Frontmatter、链接、术语、标题、代码块等必须遵守的规则。  
2. 将 **Documentation Style Guide** 从 *Getting‑Started* 移至独立 *Documentation* 区块，重命名为 **Style Guide**；同步更新了 `docs/index.yml`、`fern/docs.yml`，并添加 dev‑scope 重定向。  
3. 在 `docs/README.md` 中补充了系统构建/发布流程说明及页面搬迁时的 redirect 规则。  
4. 细化了 `fern/docs.yml` 中的 redirect 条目，确保仅在 `dev` 前缀下重定向，避免破坏 `Latest` 与固定版本的 URL。  

**🎯 影响范围**  
- **文档生成系统**（Fern）——新增/修改的 `docs/*.md`、`.mdx`、`fern/docs.yml`、`docs/index.yml`。  
- **CI 检查**：`fern check`、`fern docs broken-links`、`docs/recipes/_catalog/validate.py`。  
- **站点导航**：`docs/index.yml` 结构变更，影响侧边栏、搜索路径。  
- **页面链接**：所有对 Style Guide 的内部链接需更新为新路径。  

**💡 关注建议**  
1. **CI 通过**：在本地运行 `fern check`、`fern docs broken-links` 以及 `python3 docs/recipes/_catalog/validate.py`，确保新文档、catalog 和 redirects 均满足校验。  
2. **Redirect 正确性**：只添加 `dev` 前缀的重定向；审查 `fern/docs.yml`，防止误写到 `/dynamo/latest/` 或根路径。  
3. **文档一致性**：迁移后的 Style Guide 与其他页面的 `path`、`title` 必须保持同步，尤其是 `docs/index.yml` 中的 `- page:` 条目。  
4. **团队同步**：通知所有文档作者更新引用（如 `README.md`、issue 模板）以及 CI 脚本中可能硬编码的旧路径。  
5. **后续维护**：如再次调整导航或重命名页面，务必重复上述 redirect 流程，并在 `docs/README.md` 中记录变更历史。  

总体而言，此次变更提升了文档质量标准和可维护性，但对已有文档链接与 CI 检查提出了更严格的要求，需在合并前完成本地验证。
